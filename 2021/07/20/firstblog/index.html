我的第一篇博客

Windows系统使用hexo+github搭建个人博客

工具准备

1.npm node.js http://nodejs.cn/
2.Git 代码托管平台： https://github.com/ https://gitee.com/
3.markdown 文档编辑工具：Visual Studio Code

基本步骤

1.安装hexo
2.使用hexo命令创建网站项目目录
3.使用hexo命令创建文档，并编辑
4.使用hexo命令进行网站的本地预览
5.创建github仓库，配置文件，并将代码推送到仓库
6.使用域名远端访问
